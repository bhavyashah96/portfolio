Box Strapping Machine (Table Model)

Description:

To strap corrugated boxes wooden cases, bundles, Crates etc. and also for unitizing the products most efficient with speed upto 2.5 sec. Per cycle with minimum power consumption for 6,9,12 & 15 mm width with HD/PP Strapping. Saves Time, Manpower & Wastage of strapping

Specification:

Strapping Speed 2.5 sec/strap
Min. package size 60mm (W)
Max. package size Any size
Strap width 6,9,12,15 mm
Power Single Phase 220v Ac
Driving motor 2KW
Dimension (W x D x H ) 910 x 580 x 790 mm
Net wgt. 100Kg

Operation :

  • Connect the power plug to the power source
  • The motor starts when the switch on the Control Panel Board is Turned ON
  • The length of the band to be fed can be preset automatically by adjusting the knob meant for it
  • Set required temperature by adjusting the control knob meant for it.
  • To pull the strap back in the machine, push the Reset Switch
  • The free end of the band appears on the table. The operator need only pass the free end around the article and insert into the slot

Maintainance :

  • Never leave the machine uncleaned after use.
  • Clean the edges of the slide and the grooves in the base guide
  • Apply a few drops of light lubricant to oil holes & on sliding part prior to and after daily operations.
  • Change the oil of the reduction gear base once every year. 150ml. Should be used

Band Specification :

  • Inside diameter of paper core : 200 mm
  • Spool Width : 200 mm
  • Band width : 9mm, 12 mm & 15mm
